# Red Card 2018 - China Digital Football Awards ⚽

This repository contains the web implementation of the **Red Card 2018: China Digital Football Awards** design, developed as part of the **FentixTech Web Development Internship [Batch 2]** task. The project focuses on creating a pixel-perfect, responsive layout using advanced **CSS Flexbox** techniques.

## 📋 Project Overview

The task involved replicating two high-fidelity design screens:
1.  **Submission Interface:** A dual-panel layout with a promotional sidebar and a functional nomination form.
2.  **Results Showcase:** A structured report layout displaying award winners and club categories in an organized grid.

## 🚀 Key Features

-   **Flexbox Layout:** Implemented a fully flexible architecture for both the split-screen submission form and the dynamic results grid.
-   **Pixel-Perfect Accuracy:** Closely matched typography, brand colors (#8B1A1A), and element spacing as per the provided design assets.
-   **Responsive Design:** Optimized for various viewport sizes, including horizontal scrolling for club badges.
-   **Clean Implementation:** Structured HTML5 and modular CSS for maintainability.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for accessible web content.
* **CSS3:** Custom styling, including transitions and Flexbox.
* **Assets:** Integrated official brand imagery (`logo.png`, `bvb.png`).
